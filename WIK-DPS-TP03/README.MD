# tp-wik-dps-01

Exécuter la commande "docker build -f ./WIK-DPS-TP02/dockerfile2 -t api ." puis "docker run -it api".

### example CMD/PowerShell :
 > docker compose -f .\WIK-DPS-TP03\docker-compose.yml up

Enfin, entrez l'une des adresses IP proposées sur le terminal ou localhost:8080 et suivie de /ping sur un navigateur pour récupérer le JSON.

### example Google:
 > http://localhost:8080/ping

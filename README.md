# tp-wik-dps-01

Exécuter la commande "python.exe .\main.py" suivie du port que vous souhaitez rejoindre pour lancer l'API.

### example CMD/PowerShell :
 > python.exe .\main.py 80

Enfin, entrée 127.0.0.1: puis le port choisie et suivie de /ping sur un navigateur pour récupérer le JSON.

### example Google:
 > http://127.0.0.1:80/ping

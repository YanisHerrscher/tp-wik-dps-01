# tp-wik-dps-01

Exécuter la commande "docker build -f ./WIK-DPS-TP02/dockerfile2 -t api ." puis "docker run -it api".

### example CMD/PowerShell :
 > docker build -f ./WIK-DPS-TP02/dockerfile2 -t api .

 > docker run -it api

Enfin, entrée 127.0.0.1: puis le port choisie et suivie de /ping sur un navigateur pour récupérer le JSON.

### example Google:
 > http://127.0.0.1:80/ping
